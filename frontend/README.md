# Eioku Frontend

## Development

### Setup
```bash
npm install
```

### Run
```bash
npm run dev
```

### Test
```bash
npm test
```

### Build
```bash
npm run build
```

### Lint
```bash
npm run lint
```

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and building
- **Vitest** for testing
- **React Testing Library** for component testing
- **ESLint** for code quality
